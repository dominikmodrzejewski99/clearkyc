Foundations

Color

A clinical, cool-gray foundation. The interface should read as authoritative back-office instrumentation — closer to a medical record or trading terminal than a consumer app. Color is reserved almost entirely for the three terminal decisions; everything else lives on the neutral ramp.

Foundations

Typography

IBM Plex Sans for the entire interface — a humanist grotesque built for technical, institutional contexts that stays crisp at 11–13px. IBM Plex Mono is reserved for quoted source text from the document, so citation snippets are visually distinct from analyst-facing form values at a glance.

Each row is a label / value grid. Citations attach as a superscript footnote marker (--citation-marker) that expands the quoted source snippet in --font-mono. Overrides shift the row to a tinted accent background with an inset left bar and a mandatory, attributed justification note.

c · Terminal decision bar

Compliance decision
18 fields · 1 override · 1 missing required field — writes to immutable audit record

Default state is outlined; the analyst's current selection becomes a solid fill (shown: Escalate). The selected decision is the only saturated element in the field, so the terminal action is unmistakable. Click to preview the active states.

Inline form anchored to the field being overridden. Submission is blocked until a non-empty justification is entered; on save the field row enters the Overridden state and the note is timestamped and attributed.
